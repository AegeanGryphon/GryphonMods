ScanIndicator

Gameplay QoL

Displays scan-progress diamonds on battle animon cards showing your wiki knowledge level at a glance. No more opening the wiki mid-battle just to check whether you've already caught this one.

v1.0.1 changelog: Fixed diamond positioning on single-enemy battle cards.
v1.0.1 · AegeanGryphon GitHub ↗
🛹

HoloboardHoloken

Gameplay

Allows you to throw your holoken while riding the holoboard. Stay mobile and engage land animon without ever having to dismount. Full holoboard movement is preserved throughout the throw.

v1.0.0 · AegeanGryphon GitHub ↗
⚡

DualTypeHoloken

Gameplay

Lets an animon's hidden secondary type activate matching elemental overworld objects, not just the primary type. Unlocks the full potential of dual-type animon in the overworld.

v1.0.0 · AegeanGryphon GitHub ↗
📖

EvoScan

QoL

Automatically grants full research level 3 to an animon's evolved form when it evolves. You worked to document it — you shouldn't lose that wiki data just because it grew up.

v1.0.0 · AegeanGryphon GitHub ↗
🐛

QuestRewardFix

New
Bug Fix

Fixes a vanilla bug introduced in the second hotfix patch where a completed secret room quest re-completes four times on every save load, incorrectly granting 1,000 bonus EXP each session.

v1.0.0 · AegeanGryphon GitHub ↗
⚗️

QuickCraft

New
QoL

Dramatically speeds up fountain crafting. The vanilla game requires holding the craft button for roughly one second per item. This mod reduces that to a fraction of a second, making bulk crafting actually viable.

v1.0.0 · AegeanGryphon GitHub ↗
🌤️

WeatherWheel

Beta
Gameplay

Adds a radial weather selection wheel to the overworld. Press C (or right stick click on controller) to open it, pick a weather, and it pins to the current map. Story weather is detected and respected.

⚠️ Available to beta testers only via the launcher.
v1.0.0 · AegeanGryphon GitHub ↗
